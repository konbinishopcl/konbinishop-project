<objective>
Implementar **COM-04: Pago de suscripción** — endpoint dedicado `/subscriptions/confirm` (GET + POST) que recibe el callback de Transbank después de un pago de suscripción, confirma el pago, crea la `Subscription` row con ciclo de 30 días y créditos copiados desde Settings, marca el Order como PAID, y emite notificación `SUBSCRIPTION_ACTIVATED`.

**Purpose:** Cierra el flujo de suscripción end-to-end. Sin este plan, POST /subscriptions inicia un pago pero la Subscription nunca llega a existir.

**Output:**
- `SubscriptionsController` con endpoints `GET /confirm` y `POST /confirm` (sin auth — callback público de Transbank)
- `SubscriptionsService.handleConfirmCallback(tokenWs?, tbkToken?)` que retorna una URL de redirect al frontend
- Idempotencia: callback duplicado para Order ya PAID devuelve success sin duplicar Subscription
- Notificación `SUBSCRIPTION_ACTIVATED` emitida fire-and-forget
</objective>

    ```typescript
    async handleConfirmCallback(tokenWs?: string, _tbkToken?: string): Promise<string> {
      const frontendUrl = this.config.get<string>('FRONTEND_URL', 'http://localhost:3000');

      // Caso 1: usuario abortó en Transbank (TBK_TOKEN presente, no token_ws)
      if (!tokenWs) {
        this.logger.warn('Subscription callback without token_ws (aborted)');
        return `${frontendUrl}/cuenta/suscripcion?status=aborted`;
      }

      // Caso 2: buscar Order por externalId
      const order = await this.prisma.order.findFirst({
        where: { externalId: tokenWs },
        include: { items: true },
      });

      if (!order) {
        this.logger.error(`No subscription order found for Transbank token: ${tokenWs}`);
        return `${frontendUrl}/cuenta/suscripcion?status=not_found`;
      }

      // Validar que es un Order de suscripción (tiene al menos 1 item SUBSCRIPTION)
      const subItem = order.items.find((i) => i.type === 'SUBSCRIPTION');
      if (!subItem) {
        this.logger.error(`Order ${order.id} is not a subscription order`);
        return `${frontendUrl}/cuenta/suscripcion?status=invalid`;
      }

      // Idempotencia: si Order ya está PAID y la Subscription existe, no duplicar
      if (order.status === 'PAID') {
        const target = order.orgId ? { orgId: order.orgId } : { userId: order.userId };
        const existing = await this.prisma.subscription.findFirst({ where: target });
        if (existing) {
          this.logger.warn(`Duplicate callback for already-paid subscription order ${order.id}`);
          return `${frontendUrl}/cuenta/suscripcion?status=success`;
        }
      }

      if (order.status !== 'PENDING_PAYMENT' && order.status !== 'PAID') {
        this.logger.warn(`Subscription callback for order ${order.id} in unexpected status: ${order.status}`);
        return `${frontendUrl}/cuenta/suscripcion?status=invalid`;
      }

      // Confirmar pago con Transbank
      const pg = this.gatewayFactory.get(GatewayType.TRANSBANK);
      const confirmation = await pg.confirm(tokenWs);

      if (!confirmation.success) {
        // D-03: pago rechazado → Order=FAILED, NO crear Subscription
        await this.prisma.order.update({ where: { id: order.id }, data: { status: 'FAILED' } });
        this.logger.warn(`Subscription order ${order.id} payment failed — code ${confirmation.responseCode}`);
        return `${frontendUrl}/cuenta/suscripcion?status=failed&code=${confirmation.responseCode}`;
      }

      // Pago exitoso — crear Subscription + marcar Order=PAID en una transacción
      const creditsTotal = await this.settings.getNum('SUBSCRIPTION_CREDITS');
      const cycleStart = new Date();
      const cycleEnd = new Date(cycleStart);
      cycleEnd.setUTCDate(cycleEnd.getUTCDate() + 30);

      const target = order.orgId
        ? { orgId: order.orgId }
        : { userId: order.userId };

      const [, subscription] = await this.prisma.$transaction([
        this.prisma.order.update({
          where: { id: order.id },
          data: { status: 'PAID' },
        }),
        this.prisma.subscription.create({
          data: {
            ...target,
            status: 'ACTIVE',
            cycleStart,
            cycleEnd,
            creditsTotal,
            creditsUsed: 0,
          },
        }),
      ]);

      this.logger.log(`Subscription ${subscription.id} ACTIVE for ${target.orgId ? `org ${target.orgId}` : `user ${target.userId}`}`);

      // Fire-and-forget notification
      this.notifications.create({
        type: 'SUBSCRIPTION_ACTIVATED',
        title: 'Suscripción activada',
        body: `Tu plan está activo hasta ${cycleEnd.toISOString().slice(0, 10)}. Tienes ${creditsTotal} créditos disponibles.`,
        payload: { subscriptionId: subscription.id, cycleEnd: cycleEnd.toISOString(), creditsTotal },
        ...(target.orgId ? { orgId: target.orgId } : { userId: target.userId }),
      });

      return `${frontendUrl}/cuenta/suscripcion?status=success`;
    }
    ```

    @Post('confirm')
    @Redirect()
    @ApiOperation({ summary: 'Callback POST de Transbank para suscripción — confirma el pago y redirige al frontend' })
    @ApiBody({
      schema: {
        type: 'object',
        properties: {
          token_ws: { type: 'string', description: 'Token de pago exitoso' },
          TBK_TOKEN: { type: 'string', description: 'Token de pago abortado por el usuario' },
        },
      },
    })
    async confirmPost(
      @Body('token_ws') tokenWs?: string,
      @Body('TBK_TOKEN') tbkToken?: string,

      const url = await this.subscriptions.handleConfirmCallback(tokenWs, tbkToken);
      return { url, statusCode: 302 };

    @Get('confirm')
