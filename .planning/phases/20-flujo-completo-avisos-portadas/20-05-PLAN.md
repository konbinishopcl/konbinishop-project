Fetch pattern (apps/website/app/dashboard/sections/ArticlesSection.tsx:268-300):
```typescript
const fetchArticles = useCallback(async (p, ps, filter, signal?) => {
  if (!token) return;
  setLoading(true);
  try {
    const statusParam = STATUS_API[filter];
    const params = new URLSearchParams({ page: String(p), pageSize: String(ps), ...(statusParam ? { status: statusParam } : {}) });
    const r = await fetch(`/api/articles?${params}`, { headers: { Authorization: `Bearer ${token}` }, signal });
    if (!r.ok) throw new Error("Error al cargar artículos");
    const data = await r.json();
    setArticles(data.items ?? []);
    setTotal(data.total ?? 0); setTotalPages(data.totalPages ?? 1);
  } catch (ex) {
    if (ex instanceof Error && ex.name === "AbortError") return;
    toast.error(ex instanceof Error ? ex.message : "Error al cargar artículos");
  } finally { setLoading(false); }
}, [token]);
useEffect(() => { const c = new AbortController(); fetchArticles(page, perPage, activeFilter, c.signal); return () => c.abort(); }, [fetchArticles, page, perPage, activeFilter]);
```

Action pattern (ArticlesSection.tsx:316-358): each action fetches the PATCH/DELETE,
toasts, then calls fetchArticles again to refresh.
