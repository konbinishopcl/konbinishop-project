Backend response shapes (verified against source):

GET /users (UsersService.findAll, USER_SELECT) returns an ARRAY (no pagination wrapper):
```typescript
// each item:
{ id, email, firstname, lastname, rut, isCompany, role, confirmed, blocked,
  type: 'PERSON' | 'ORGANIZATION', handle, isVerified, createdAt, updatedAt }
```

PATCH /users/:id/ban  body { blocked: boolean }  → returns updated user (SUPER_ADMIN only).

GET /faq (FaqService.findAll) → ARRAY of { id, question, answer, order, createdAt, updatedAt }.
POST /faq          body { question, answer, order? }  (ADMIN+)
PATCH /faq/:id     body { question?, answer?, order? } (ADMIN+)
DELETE /faq/:id    → (ADMIN+)

GET /admin/audit-logs (AuditService.findAll) → { items, total, page, pageSize, totalPages }.
  query params: page?, pageSize?, action?, entity?, userId?, dateFrom?, dateTo?
  each item is a RAW AuditLog scalar row (NO joins):
  { id, userId: number|null, action: 'CREATE'|'UPDATE'|'DELETE'|'APPROVE'|'REJECT'|'BAN'|'UNBAN',
    entity: 'EVENT'|'USER'|'AVISO'|'PORTADA', entityId: number, metadata: object,
    ip, userAgent, url, createdAt }

GET /services/photography/options       → ARRAY of { id, label, order }
GET /services/content-creators/options  → ARRAY of { id, label, order }
POST   /services/{photography|content-creators}/options       body { label, active?, order? } (ADMIN+)
PATCH  /services/{photography|content-creators}/options/:id    body { label?, active?, order? } (ADMIN+)
DELETE /services/{photography|content-creators}/options/:id    (ADMIN+)
