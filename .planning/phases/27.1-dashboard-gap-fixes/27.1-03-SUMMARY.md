# Phase 27.1 Plan 03: Real KPIs + Category Bars + Ver toda link in HomeSection Summary

HomeSection KPIs and category panel wired to real API data — INGRESOS MES from current-month PAID payments, EVENTOS PUBLICADOS from approved events total, SUSCRIPTORES from subscriptions total, category bars from eventCategories() _count.events, revenue chart bucketed from real payments, and Ver toda navigates to /dashboard/events.

## What Was Built

Replaced all hardcoded mock data in HomeSection with real API-driven values:

- **INGRESOS MES KPI**: Sum of current-month PAID payments via `api.adminPayments(token)` filtered by year/month
- **EVENTOS PUBLICADOS KPI**: Real total from `api.adminEvents(token, { status: "APPROVED", pageSize: 1 }).total`
- **SUSCRIPTORES KPI**: Real total from `api.subscriptions(token).total`
- **EN REVISIÓN KPI**: Already real (pendingTotal), preserved
- **Category bars**: Top 6 categories by `_count.events` from `api.eventCategories()`, pct relative to max count
- **Revenue chart**: Monthly buckets of PAID payments for last 12 months; empty array → RevenueBarChart empty state
- **Ver toda →**: Replaced dead button with `<Link href="/dashboard/events">` using next/link
