const authorName = "Konbini Editorial";

With:
```typescript
const authorName = (() => {
  const u = article.user;
  if (!u) return "Konbini Editorial";
  const full = `${u.firstname ?? ""} ${u.lastname ?? ""}`.trim();
  return full || "Konbini Editorial";
})();
```

This handles three cases:
1. `article.user` is null/undefined → "Konbini Editorial" (imported WordPress articles)
2. user exists but firstname+lastname are both empty/null → "Konbini Editorial"
3. user exists with real names → "Gabriel Burgos" (or whatever the names are)

The existing `authorInitials(authorName)` call at line 129 and the display at line 130 require NO changes — they already use the `authorName` const.
