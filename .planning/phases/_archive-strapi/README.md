# Archivo — Roadmap Strapi (obsoleto)

Esta carpeta guarda el trabajo de planning del **stack anterior** del proyecto:
Strapi 5 + Nuxt 4 + un dashboard Next.js separado.

El 2026-05-20 el proyecto migró a **NestJS + Prisma + Next.js** y el modelo de negocio
cambió (publicar es gratis; no se venden entradas). El roadmap de pagos
(Transbank / Mercado Pago / Stripe / emails MJML) quedó sin efecto.

## Qué hay aquí

- `01-security-foundation/` — Phase 1 del roadmap Strapi: role enforcement del dashboard,
  CORS de Strapi, proxy Nuxt, reCAPTCHA, cookie JWT HttpOnly. Aplicaba a código que ya no
  existe en el repositorio.

Se conserva solo como referencia histórica. **No es parte del roadmap activo.**
El roadmap vigente está en `.planning/ROADMAP.md`.
