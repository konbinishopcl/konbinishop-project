# Quick Task d8k: Documentar la API con Swagger

**One-liner:** Agregar documentación OpenAPI/Swagger a la API NestJS — UI navegable en
`/docs` y especificación en `/docs-json`.

## Tareas

1. Instalar `@nestjs/swagger`.
2. Activar el plugin `@nestjs/swagger` en `nest-cli.json` (introspección automática de DTOs).
3. Configurar `SwaggerModule` en `main.ts` con `DocumentBuilder` (título, versión,
   `addBearerAuth`); servir la UI en `/docs`.
4. Anotar los controllers: `@ApiTags` por controller, `@ApiBearerAuth` en los endpoints
   protegidos, `@ApiOperation` con un resumen por endpoint, y `@ApiConsumes`/`@ApiBody` en
   la subida de imágenes.

## Verificación

- `nest build` compila sin errores
- `GET /docs` sirve la UI de Swagger; `GET /docs-json` devuelve el OpenAPI JSON
- Todos los endpoints (auth, users, events, catalog, uploads) figuran documentados
